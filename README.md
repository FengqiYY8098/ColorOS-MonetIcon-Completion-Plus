# ColorOS-MonetIcon-Completion-Plus
ColorOS 莫奈图标模块的非官方图标补全项目。包含自动化脚本，辅助制作 Material You 风格单色图标。

由于模块作者未授权，此项目只提供图标文件，请自行将图标添加到模块中。 欢迎各位commit图标。

适配指南

    创建一个文件夹，在文件夹内创建名为 icon 的文件夹，放置原始图标文件 (需要将原图标文件名更改成包名，文件后缀无所谓)。

    运行 mkdir.py 后，会自动根据 icon 文件夹内的文件名在当前目录下创建 uxicons 文件夹并创建包名目录，并自动生成 packages.txt 包名列表(按字母排序)。

    将绘制好的图标放入 uxicons 内对应的文件夹后执行 clean.py 会将图标转换成RGBA格式并清除图片元数据。

    执行 rename.py 后会将 uxicons 文件夹内所有文件重命名为 monochrome.png。

    此时就已经可以将 uxicons 文件夹合并到模块内开始使用了！

图标规范

    图标可参考 icons.psd。

    画布大小为 240x240 像素，图标本体取最长一边，调整为 90 像素后居中。

    莫奈图标本质上是一层遮罩，颜色(灰度/明度)取决于图标该处的透明度，而并非颜色，可通过调整图标部分区域的透明度来实现不同的灰度。

适配需求

    找到你想要求适配的应用的安装包，打开后找到 res 文件夹解压出来，将 res 文件夹重命名为对应的应用包名。将所有文件夹放到一个压缩包内并提交 issue。

    注意：提交适配前请自行检查，确保该图标未被该项目适配。
